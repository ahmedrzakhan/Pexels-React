# Pexels clone
## Tech Stack
- React
- React Router Dom
- Javascript
- Html
- Css

## Live URL
https://pexels-react.netlify.app/

## Instructions to run locally
```
git clone https://github.com/ahmedrzakhan/Pexels-React.git
cd Pexels-React
yarn install
yarn start
```
